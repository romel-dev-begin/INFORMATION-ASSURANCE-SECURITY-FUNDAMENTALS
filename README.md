# Information Assurance and Security Fundamentals

A web-based presentation for **Module 2: Information Assurance and Security Fundamentals**.  
This project presents cybersecurity concepts in an interactive and modern way using HTML, CSS, and JavaScript instead of traditional PowerPoint or Canva slides.

## Project Overview

This website-based presentation is designed to explain the basic concepts of information assurance and security. It uses a cybersecurity command center theme to make the discussion more engaging, professional, and visually appealing.

The presentation covers important topics such as Information Assurance, the CIA Triad, Security Governance, Security Policies, Risk Management, and Security Compliance.

## Topics Covered

1. **Principles of Information Assurance**
   - Meaning of Information Assurance
   - Importance of protecting information
   - Basic principles such as confidentiality, integrity, availability, authentication, authorization, and non-repudiation

2. **Confidentiality, Integrity, and Availability**
   - Explanation of the CIA Triad
   - How confidentiality protects private information
   - How integrity keeps data accurate and trustworthy
   - How availability ensures systems can be accessed when needed

3. **Information Security Governance**
   - Meaning of security governance
   - Importance of roles, responsibilities, policies, and decision-making in security

4. **Security Policies, Standards, and Procedures**
   - Difference between policies, standards, and procedures
   - Examples of password policies, access control policies, and security procedures

5. **Risk Management Fundamentals**
   - Meaning of risk management
   - Common security risks
   - Steps in identifying, analyzing, treating, and monitoring risks

6. **Security Compliance and Regulatory**
   - Meaning of compliance
   - Importance of following laws, standards, and organizational rules
   - Connection between compliance and information security

## Project Purpose

The purpose of this project is to present Information Assurance and Security Fundamentals in a more interactive and engaging format. Instead of using static slides, this project allows the presenter to use a website-style presentation that looks more modern and unique.

## How to Run the Project

1. Download or clone the project folder.
2. Open the folder on your computer.
3. Locate the `index.html` file.
4. Double-click `index.html` or open it using a web browser.
5. Navigate through the presentation using the available buttons or sections.
